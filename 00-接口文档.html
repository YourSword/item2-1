支持接口

  • 接口地址

    http://192.168.50.130:8001/support/:id

  • 请求方式

    get

  • 参数说明

参数名称 是否必须 说明
id 是 评论的id
  • 返回说明
参数 说明
code 执行结果代码，1 有评论数据，2没有评论数据
data 如果没有评论，返回字符串，如果有评论返回数组
  • 示例
{code: 0, data: "没有要支持的评论"}

发表评论接口

  • 接口地址

    http://192.168.50.130:8001/comments/create

  • 请求方式

    post

  • 参数说明

参数名称 是否必须 说明
author 是 回复的用户
author_email 否 回复用户的邮箱
content 是 回复内容
post_id 是 回复的帖子的id
  • 返回说明
参数 说明
code 执行结果代码，1成功，0失败
comment 评论的内容
  • 示例
{code:1, comment: {}}

登录接口

  • 接口地址

    http://192.168.50.130:8001/login

  • 请求方式

    post

  • 参数说明

参数名称 是否必须 说明
username 是 用户名
password 是 密码
  • 返回说明
参数 说明
code 执行结果代码，1注册成功，0注册失败
msg 结果提示信息

    msg: '注册失败'
}
